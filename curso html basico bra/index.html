Hyper Text Transfer Protocol (HTTP) é o protocolo usado pelos servidores web para transmitir documentos HTML pela internet.

File Transfer Protocol(FTP) é outro protocolo muito utilizado para transmitir arquivos, porém sua interface gráfica não é tão agradável como o HTTP.

Transfer Control Protocol(TCP) é um conjunto de protocolos para trasmissão de informação pela web, com função de verificar se os dados são transferidos de forma correta, na sequência apropriada e sem erros.

Internet Protocol(IP) é um protocolo de endereçameto, que fornece o endereço dos computadores na rede.
